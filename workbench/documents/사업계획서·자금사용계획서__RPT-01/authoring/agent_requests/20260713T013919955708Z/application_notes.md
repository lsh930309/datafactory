# 적용 메모

- 고정된 8개 `field_id`와 `semantic_path`, anchor binding은 변경하지 않았습니다.
- 각 자유기재 필드에 용도별 합성값 20개로 구성된 전용 pool을 연결했습니다. 최종 적용 기준은 `faker_profile_draft.json.field_generators`입니다.
- 전체 템플릿에서 입력 영역 오른쪽의 `)`는 정적 인쇄 요소이고 별도 단위는 없으므로, 생성값에는 괄호나 단위를 포함하지 않았습니다.
- 현재 binding은 모두 독립 자유기재 문자열입니다. 연결 가능한 금액·체크박스·날짜 field가 없어 `constraints`는 의도적으로 비워 두었습니다.
- 적용 전 UI에서 8개 pool의 문구 적합성과 bbox 내 렌더 길이를 확인한 뒤 draft를 승인하십시오. 1·2페이지의 나머지 금액·체크박스·날짜 관계는 후속 bbox review와 schema 확장이 필요합니다.

공식 웹 근거는 2026-07-13에 재확인했습니다. 자산형성포털은 이 서식을 적립 지원금의 지원목적별 사용예정 금액을 적는 문서로 설명하며, 보건복지부는 희망저축계좌Ⅱ·청년내일저축계좌의 지급 요건으로 제출을 안내합니다.
