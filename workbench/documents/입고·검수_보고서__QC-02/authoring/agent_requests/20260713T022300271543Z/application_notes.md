# QC-02 2차 authoring 초안 메모

- 고정된 `schema_draft.json`, `anchor_map_draft.json`, `research_report.json`, `stylesheet_draft.json`은 변경하지 않았습니다.
- 72개 binding 모두에 지원 generator를 정확히 하나씩 지정했습니다.
- 품명·규격은 행별 24개 합성 record pool에서 함께 선택하고, 발주일과 입고일은 유효 날짜·선후관계·기준일(2026-07-13) 상한을 적용했습니다.
- 원본의 날짜 bbox에는 `20`이 보이지만 인페인트 이미지에서는 제거됩니다. 고정 schema의 `static_prefix=20` 계약을 따라 연도는 뒤 2자리로 생성하며, 적용 전 샘플 렌더 확인이 필요합니다. `년`, `월`, `일`, `(인)`은 생성값에 중복하지 않습니다.
- 품명·규격·품질·기타·비고는 같은 행의 합성 검사 record에서 함께 선택합니다. 수량은 별도 1~999 범위이며, 불량수량 bbox가 없어 산술 분리는 하지 않습니다.

적용 전 UI에서 `기타` 열의 조직 표준 선택지와 상단 결재 단계 명칭을 검토하십시오. 승인 후 draft 4개를 함께 적용하면 됩니다.
